                                                                                                                                      Tamas Novak Piker • 4 years ago

                                                                                                                                      putchar C function writes to stdout. This font stuff is for embedded systems, where usually there is no operating system, so no any "stdout" device. If you use a microcontroller, then a bitmap LCD is interfaced to your MCU (e.g. D0..D7 bus, R/W, RS, E lines). You must tick-tock these lines from program to send pixel data to LCD. And at first LCD must be sent an initialization sequence. Check reference of LCD module, and try to understand the concept...I can send you sample C source to Microchip PIC and ST ARM MCU's.
